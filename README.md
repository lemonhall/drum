# 太鼓达人 - Godot 4 版本

这是一个使用 Godot 4 制作的太鼓达人风格的音乐节拍游戏。

## 游戏玩法

- **红色音符 (咚)**: 按 `Z` 或 `X` 键击打
- **蓝色音符 (咔)**: 按 `C` 或 `V` 键击打
- 音符从左侧出现，向右移动到击打区域（橙色圆圈）
- 在音符到达击打区域时按下对应按键来击打音符
- 击中音符获得分数，连续击中可以获得连击奖励

## 游戏特性

- ✅ 基本的音符生成和移动
- ✅ 两种音符类型（红色DON，蓝色KA）
- ✅ 击打判定系统
- ✅ 分数和连击系统
- ✅ 简单的UI界面
- ✅ 键盘输入控制

## 文件结构

```
├── scenes/
│   ├── GameMain.tscn     # 主游戏场景
│   └── Note.tscn         # 音符场景
├── scripts/
│   ├── GameMain.gd       # 游戏主控制器
│   ├── Note.gd           # 音符类
│   ├── NoteSpawner.gd    # 音符生成器
│   └── DrumController.gd # 太鼓控制器
└── project.godot         # 项目配置
```

## 如何运行

1. 在 Godot 4 中打开项目
2. 运行主场景 `scenes/GameMain.tscn`
3. 使用键盘按键击打音符
